Create a package.json file in your project directory.Після установки  node.js в проекті запускаєм команду - npm init. Після чого буде список питань де можна донастроєти проекти і після кожної відповіді ентер що перейти до наступного питання. Або ентер на всі питання і все буде діфолт і зявиться package.json.
Далі можна наповняти цей файл пакетами і один із них gulp.
npm install --global gulp-cli(npm install gulp-cli -g) - для роботи з gulp в терміналі. --global - дає можливість робити глобально з будьякої папки з будьяким проектом тобто без цогофлага він встановиться в тільки в цю папку для цього проекта.
Після встановлюєм локально в цей проект через npm install --save-dev  gulp-cli
Install the gulp package in your devDependencies - npm install --save-dev gulp

Далі при виборі пакета на https://www.npmjs.com/package/ в кінці кожного пакета для його збереження в поточний проект в кінці --save-dev  н-д: 
npm i gulp-sass --save-dev

Для запуска просто gulp - це діфолт тобто в цій команді буде прописаний список задач  які і будуть атоматом виконуватись.
Для окремих конкретних задач н-д: gulp server - запуститься тільки сервер 
Для зупинки сервера CTR+c

Якщо перенести файли gulpfile.js і package.json в інший проект то команда
(npm i) заінсталірує всі файли
